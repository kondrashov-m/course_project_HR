# Информационно-справочная система отдела кадров предприятия

## Введение

Данная система предназначена для управления сотрудниками предприятия, их отделами, должностями и зарплатой. Она позволяет:

- Добавлять, редактировать и удалять сотрудников.
- Управлять отделами.
- Расчитывать и генерировать отчеты.
- Сохранять данные (в памяти или в файле).

## Анализ требований

### Функциональность

1. Управление списком сотрудников:
   - Добавление, редактирование, удаление.
   - Просмотр всех сотрудников.

2. Управление отделами:
   - Добавление, редактирование, удаление.
   - Привязка сотрудников к отделам.

3. Расчет зарплаты:
   - Оклад, премии, отпуска.
   - Генерация отчета.

4. Валидация данных.
5. Сохранение данных.

## Проектирование

Система реализована на основе ООП:

- Класс `Employee` — сотрудник.
- Класс `Department` — отдел.
- Класс `Position` — должность.
- Класс `Salary` — зарплата.
- Классы-сервисы для работы с данными.
- Репозитории — для хранения данных.

## Реализация

Каждый класс реализован в отдельном файле с соответствующим именем.

## Тестирование

Проведены тесты на добавление, удаление, редактирование сотрудников.

## Заключение

Система успешно реализована. Использованы принципы ООП. Можно расширить функциональность (например, добавить отчеты, импорт/экспорт).
