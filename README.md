# HRSystem — система учёта сотрудников

Коротко: консольное приложение для хранения и просмотра информации о сотрудниках, отделах, должностях, зарплатах, контрактах и отпусках.

## Возможности

- Добавление и просмотр сотрудников и их данных.
- Управление отделами и должностями.
- Учёт зарплат и ведомостей.
- Хранение информации о контрактах и отпусках.
- Удобный консольный интерфейс с рамками и выравниванием.

## Структура проекта и за что отвечает каждый файл

- `Program.cs`: точка входа приложения, инициализация сервисов и запуск меню.
- `App.config` / `bin/.../App.config`: конфигурация приложения (при необходимости).

- Папка `Commands/`:
   - `ICommand.cs`: интерфейс для команд меню.
   - `EmployeeCommand.cs`, `DepartmentCommand.cs`, `SalaryCommand.cs`, `VacationCommand.cs`: реализации команд пользовательского интерфейса.

- Папка `Models/` — модели доменной области:
   - `Employee.cs`: модель сотрудника.
   - `Department.cs`: модель отдела.
   - `Position.cs`: модель должности.
   - `Salary.cs`: модель записи о зарплате.
   - `Vacation.cs`: модель отпуска.
   - `Contract.cs`: модель трудового договора (дата начала/окончания, тип договора).

- Папка `Repositories/` — репозитории для хранения данных:
   - `IEmployeeRepository.cs`: интерфейс репозитория сотрудников.
   - `BaseRepository.cs`, `InMemoryRepository.cs`, `EmployeeRepositoryEx.cs`: реализации репозиториев (в памяти и расширенные).

- Папка `Services/` — бизнес-логика и сервисы:
   - `IEmployeeService.cs`, `IDepartmentService.cs`, `ISalaryService.cs`, `IVacationService.cs`: интерфейсы сервисов.
   - `EmployeeService.cs`, `DepartmentServiceImpl.cs`, `SalaryService.cs`, `VacationService.cs`: реализации сервисов.

- Папка `Utils/` — утилиты для работы с консолью и вводом:
   - `ConsoleHelper.cs`: обёртка над `Console` с поддержкой чтения Юникода и методами для вывода рамок.
   - `InputValidator.cs`, `IInputValidator.cs`: валидация ввода пользователя.
   - `MenuManager.cs`: логика отображения меню и взаимодействия с пользователем.

## Замечания по коду

- Удалены испорченные комментарии и исправлены проблемы с кодировкой в некоторых файлах (например, `Models/Contract.cs`).
- Добавлен метод вывода рамки в `Utils/ConsoleHelper.cs` для более целостного интерфейса.

## Как запустить

Скомпилировать и запустить через `dotnet` (в Windows):

```powershell
dotnet build
dotnet run --project HRSystem.csproj
```
